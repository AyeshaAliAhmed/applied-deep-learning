# Applied Deep Learning in Physics

## Course Overview
This repository contains my coursework, exercises, and projects from the 
**Applied Deep Learning in Physics** course (2nd Semester, University).

The course focuses on state-of-the-art deep learning techniques applied 
to real-world physics problems, using the TensorFlow/Keras framework.

## What I Learn & Build
- Fundamentals of Deep Learning (Generalization, Regularization, Optimization)
- Classification and Regression Tasks
- Convolutional Neural Networks (CNNs)
- Recurrent Neural Networks (RNNs)
- Autoencoders
- Advanced Computer Vision Methods
- Applications in Physics:
  - Image recognition
  - Time series data analysis
  - Pulse shape discrimination
  - Real-time low-power on-device computing

## Tools & Frameworks
- Python
- TensorFlow / Keras
- Jupyter Notebooks

## Learning Outcomes
By the end of this course I am able to:
- Summarize and apply core deep learning concepts
- Design and optimize neural network architectures for physics problems
- Verify model results against experimental data

## Repository Structure
Each folder represents a separate project or exercise from the course.
